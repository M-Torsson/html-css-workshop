# Tech Store Website

## Project Description
Tech Store is a simple multi-page website built using HTML and CSS.
The project includes a login page, home page, products page, order form, and contact page.

This project was created as part of an HTML & CSS workshop.

---

## Login Information
The login page is for demonstration purposes only.
You can enter any email ending with `@gmail.com` and any password to access the store pages.
No real authentication or database is used in this project.

---

## Technologies Used
- HTML5
- CSS3
- Flexbox Layout
- Google Fonts (Cairo)

---

## Pages Included
- Login Page (index.html)
- Home Page (store.html)
- Products Page (products.html)
- Order Page (order.html)
- Contact Page (contact.html)

---

## Features
- Responsive product card layout
- Image overlay design on home page
- Active navigation highlight
- Custom cart icon with hover and click effects
- Sticky footer layout

---

## How to Run
Open `index.html` in your browser or use Live Server in VS Code.
